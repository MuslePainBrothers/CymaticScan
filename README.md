# CymaticScanApp
## 概要
性格診断メーカー

## 機能
* 幾つかの質問に答えて、ユーザの性格を診断する.
* 質問から評価値を計算して,診断結果をユーザに伝える.

## 開発環境
* 言語 python
* フレームワーク django
